CSS IDs

An ID is a UNIQUE identifier

ID Notes

  • In CSS an ID starts with an octothorpe (pound, hashtag, number sign) - #
  • Example uses for IDs: Target a specific header, section, or a particular item.
  • Each ID should only be used once per page - they are UNIQUE.
  • Used for one-off styling.
